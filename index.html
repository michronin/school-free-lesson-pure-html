למה אני צריך לקבל מרחב למידה אישי בזמן שיעור מדעי המחשב?
  • קיבלתי 99 במבחן, ואני בטוח שתוכל להסכים שזהו ציון ממש גבוה. הייתה לי טעות אחת, על משתנים מוחלטים ומשתנים יחסיים, והתבלבלתי, מכיוון שהמוח שלי עבד מהר יותר מהיד שלי, ולא הקפתי את התשובה שהתכוונתי להקיף, למרות שידעתי את התשובה.
  • ניתן לראות, שאני יודע לתכנת, מעצם העובדה שאתה עכשיו באתר הזה ורואה את התוכנה שלי והפרוייקים שלי.
  • בנוסף, אני יודע לתכנת ב-6 שפות:
    1. JavaScript - אני מתכנת משחקים ואתרים ב-JavaScript, חלק מן האתר הזה, בנוי על JavaScript.
    2. HTML - אני מתכנת משחקים ואתרים בעזרת HTML, ורב האתר הזה בנוי על HTML.
    3. CSS - אני מעצב, ויוצר משחקים בעזרת CSS. אני מאתגר אותך לקרב CSS 😀.
    4. React Native - אני יוצר אפליקציות לטלפון באמצעות React native.
    5. NodeJS - אני יוצר אתרים בעזרת NodeJs, עם בסיס נתונים Mongo DB. יצרתי כמה אתרים מעניינים עם NodeJs, אך איני יכול לצרף אותם פה.
    6. Java - אני לומד לתכנת ב-Java, ויש לי כבר מספר פרוייקטים מעניינים שלא אוכל לצרף פה 😥.
